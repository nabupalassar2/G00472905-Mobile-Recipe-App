# Mobile Recipe Application

Student Number: G00472905  
Module: Mobile Applications Development  
Year: 2025

## Overview
This mobile application allows users to search for recipes by ingredients,
view recipe details, manage favourite recipes, and switch between Metric and
US measurement units.

The application is built using Ionic and Angular.

## Features
- Search recipes by ingredients
- View recipe details (ingredients and instructions)
- Add and remove recipes from favourites
- Persistent favourites using Local Storage
- Measurement units toggle (Metric / US)


## Pages
- Home Page (recipe search)
- Recipe Details Page
- Favourites Page
- Settings Page

## Data Storage
- Favourite recipe IDs are stored using browser Local Storage
- Measurement unit preference is stored persistently

## Innovation
A quick favourite toggle is available directly on the Home Page,
allowing users to add or remove recipes from favourites without
opening the recipe details page.
